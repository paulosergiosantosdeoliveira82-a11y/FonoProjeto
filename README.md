📡 Dispositivo Portátil para Estímulo à Comunicação Oral com ESP32

🧠 Visão Geral

Este projeto apresenta o desenvolvimento de um sistema embarcado portátil de baixo custo, baseado no microcontrolador ESP32, com o objetivo de auxiliar no estímulo à comunicação oral em contextos de reabilitação fonoaudiológica e tecnologias assistivas.

A proposta integra Led, botões, atuadores e comunicação sem fio via Bluetooth, permitindo a interação com dispositivos externos e contribuindo para práticas terapêuticas mais acessíveis e tecnológicas.

---

🎯 Objetivos

Objetivo Geral

Desenvolver um dispositivo portátil inteligente para auxiliar no estímulo à comunicação oral.

Objetivos Específicos

- Implementar comunicação Bluetooth com dispositivos externos
- Desenvolver lógica embarcada para estímulo interativo
- Criar uma solução de baixo custo e fácil replicação

---

⚙️ Funcionalidades
- 📶 Comunicação via Bluetooth
- 🔔 Feeback Multimoldal
- 🔋 Operação portátil

---

🧰 Tecnologias Utilizadas

- ESP32
- Linguagem C/C++ (Arduino Framework)
- Bluetooth ( A2DP/AUDIOTOOLS)
- Modelagem e Impressão Tridimensional
---

🏗️ Arquitetura do Sistema

O sistema é composto pelas seguintes  camadas principais:

1. Processamento
   ESP32 executando a lógica de controle e tomada de decisão

2. Comunicação
   Transmissão de dados via Bluetooth para dispositivos externos

🔌 Hardware Utilizado

- ESP32
- Modulo Amplificador de audio Lm35
- Alto-falante
- Motores vibratorios Vibracall1027
- Potenciômetro de 10Kohm's
- Resistores de 100ohm's
- Diodos emissores de luz (LED's)

- Chaves de duas posições on/off
